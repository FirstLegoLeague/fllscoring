FLL Scoring Demo App

Welcome to the demo-environment of the FLL scoring app developed by several Dutch FLL volunteers.

This demo demonstrates several important features derivied from these requirements including:

  • a responsive design
  • the generation of the scoring file from an XML file
  • and options to communicate with other modules

DO NOT
use this page during live events: everyone can manipulate the data and the environment will be reset every 30 minutes.

However, feel free to experiment and test. If you want your own copy to use at your event, or if you like to give some feedback or check out other cool modules (like this display system), feel free to visit our GitHub

Have FUN!
